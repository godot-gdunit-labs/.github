## About Godot GdUnit Labs

Godot GdUnit Labs is an open-source initiative dedicated to providing comprehensive, high-performance testing solutions for the Godot Game Engine.
Our mission is to enhance game development quality through robust, easy-to-use unit testing frameworks that support GDScript and C# programming languages and integrate seamlessly with modern development workflows.
We focus on:
- Developing cutting-edge testing tools
- Supporting both GDScript and C# testing
- Providing seamless CI/CD integration
- Improving game development reliability

## Projects

### [GdUnit4](https://github.com/godot-gdunit-labs/gdUnit4)
A powerful and easy-to-use Unit Testing Framework for the Godot Game Engine.

- Comprehensive testing solution for Godot game development
- GDScript and CSharpScript support

### [GdUnit4Net](https://github.com/godot-gdunit-labs/gdUnit4Net)
A feature-complete unit testing framework for Godot C# projects with VSTest compatibility.

- Full integration with Visual Studio Test Platform
- Compatible with .NET 8.0 and 9.0

### [GdUnit4Net Examples](https://github.com/godot-gdunit-labs/gdUnit4NetExamples)
Comprehensive example projects demonstrating advanced GdUnit4Net testing patterns and configurations.

- Practical testing scenarios
- Best practices demonstrations
- Learning resource for developers

### [GdUnit4 GitHub Action](https://github.com/godot-gdunit-labs/gdUnit4-action)
Continuous Integration tool for seamless GdUnit test automation in GitHub workflows.

- Automate game testing
- Integrate with CI/CD pipelines
- Simplify testing processes
